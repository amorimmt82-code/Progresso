# Exportação de Dados para Excel

Aplicação web desenvolvida para gerar e exportar informações para ficheiros Excel, permitindo organizar e disponibilizar dados de forma rápida e prática.

## Funcionalidades

* Exportação de dados para ficheiros Excel (.xlsx);
* Geração automática de planilhas;
* Interface simples para seleção e exportação de informações;
* Processamento dos dados antes da exportação;
* Layout responsivo.

## Tecnologias utilizadas

* React
* TypeScript
* Vite
* Express
* Tailwind CSS
* XLSX (SheetJS)

## Instalação

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

Entre na pasta do projeto:

```bash
cd SEU-REPOSITORIO
```

Instale as dependências:

```bash
npm install
```

Caso necessário, configure as variáveis de ambiente no ficheiro `.env`.

Inicie a aplicação:

```bash
npm run dev
```

## Estrutura do projeto

```text
src/
public/
server/
components/
utils/
```

## Objetivo

O objetivo deste projeto é facilitar a geração e exportação de dados para o formato Excel, permitindo criar planilhas organizadas para utilização em relatórios, controlo de informação ou integração com outros processos.
